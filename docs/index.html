Gmail MCP Bridge

Control Gmail from Claude Desktop via a Chrome Extension + Model Context Protocol (MCP). No API keys. Free.

GitHub Repo · Discussions · Installation · Quick Start

Why this project?

Works locally with your browser session. No Google API keys. Just install the extension, start the MCP server, and connect Claude Desktop.

Get started

  1. Install prerequisites (Node.js 18+, Chrome)
  2. Follow the Quick Start guide
  3. Join Discussions for help and ideas
